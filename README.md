# MyURLs

MyURLs is a open-source social media reference landing page.

## Installation

Use the control system [git](https://git-scm.com/downloads) to install MyURLs.

```bash
git clone https://github.com/XpertBotTeam/MyURLS.git
```

## Usage

```bash
php artisan serve
```
